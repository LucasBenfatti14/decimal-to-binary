## 🚀 Sobre o projeto

Este projeto consiste em um conversor de números decimais para binário desenvolvido em Python, com foco na implementação manual do algoritmo de conversão.

Ao invés de utilizar funções prontas da linguagem, toda a lógica foi construída do zero através de divisões sucessivas por 2 e armazenamento dos restos da divisão, simulando o funcionamento matemático da conversão binária.

Trata-se de um projeto **100% autoral**, criado com o objetivo de praticar lógica de programação, estruturas de repetição e manipulação de listas em Python.
